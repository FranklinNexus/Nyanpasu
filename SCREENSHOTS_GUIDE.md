# 📸 截图准备指南

## 为什么截图重要？

好的截图能让用户：
- 🎯 快速理解应用功能
- 💡 看到实际使用效果
- ✨ 产生下载欲望

**统计数据**：好的截图可以提升 30-50% 的转化率！

---

## 📱 技术要求

### Google Play 要求
- **最少数量**：2 张
- **推荐数量**：4-8 张
- **尺寸**：1080 x 2400 px（或 1440 x 3120 px）
- **比例**：9:20（现代手机屏幕比例）
- **格式**：PNG 或 JPEG
- **文件大小**：每张不超过 8MB

## 🎯 截图最佳实践

### ✅ 做什么
- **使用真实内容** - 展示实际运行效果
- **高质量图片** - 确保预览图是精美的壁纸
- **多样性** - 展示不同功能和场景
- **清晰文字** - 如果添加标注，字体要大且清晰
- **统一风格** - 所有截图使用相同的手机/主题

### ❌ 不要做什么
- **不要使用模拟器** - 用真机截图更真实
- **不要过度 PS** - 保持真实性
- **不要太多文字** - 让画面说话
- **不要低质量图** - 模糊、压缩过度的截图很丑
- **不要包含系统 UI** - 隐藏状态栏通知（可选）

## 🛠️ 截图工具推荐

### 方法 1：Android Studio（推荐）
1. 手机连接电脑
2. 运行应用
3. Android Studio → Logcat 旁边的 📷 图标
4. 自动保存为 PNG

### 方法 2：手机自带
- **Android**: 音量下 + 电源键
- **Samsung**: 侧边按钮 + 音量下
- **小米/华为**: 三指下滑

### 方法 3：ADB 命令
```bash
# 截图并保存到电脑
adb shell screencap -p /sdcard/screenshot.png
adb pull /sdcard/screenshot.png ./screenshots/
```

## 🎨 后期处理（可选）

### 添加设备框架
使用工具：
- **Device Art Generator**（Google 官方）
- **Mockuphone**（在线工具）
- **Figma**（专业设计）

让截图看起来像在真实手机上：
```
Before: 纯截图
After: 截图 + 手机边框 + 背景
```

### 添加文字标注
使用工具：
- **Canva**（在线，免费）
- **Figma**（专业）
- **Photoshop**（如果你会用）

标注建议：
- 字体：无衬线字体（如 Roboto、Noto Sans）
- 大小：至少 40px
- 颜色：与应用主题一致（粉色/蓝色）
- 位置：顶部或底部，不遮挡重要内容

## 💡 专业提示

### 让截图脱颖而出

1. **第一张最重要**
   - 用户首先看到的
   - 展示最吸引人的功能
   - 考虑添加 "NEW!" 或特色标签

2. **讲一个故事**
   - 按照用户使用流程排序
   - 从下载 → 设置 → 查看效果

3. **展示独特卖点**
   - 看板娘互动（其他应用没有的）
   - 双模式（差异化功能）
   - 极简设计（审美优势）

4. **考虑 A/B 测试**
   - 上传后观察哪些截图被点击最多
   - Google Play Console 有相关数据
