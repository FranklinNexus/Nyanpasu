## 🎯 下一步操作清单

### 阶段 1️⃣：代码最终检查（30 分钟）

- [ ] **1.1 Sync Gradle**
  - 在 Android Studio 中点击 **"Sync Now"** 🐘
  - 等待 PhotoView 依赖下载完成

- [ ] **1.2 运行测试**
  ```bash
  # 在 Terminal 运行
  ./gradlew clean
  ./gradlew assembleDebug
  ```

- [ ] **1.3 功能测试**
  - [ ] 拖拽预览图（单指）
  - [ ] 缩放预览图（双指捏合）
  - [ ] 双击快速缩放
  - [ ] 应用壁纸后检查裁剪是否正确
  - [ ] 双壁纸模式切换
  - [ ] 看板娘对话
  - [ ] 彩蛋触发（点击 Logo 10 次）

- [ ] **1.4 修复 Linter 错误**
  ```bash
  # 检查代码质量
  ./gradlew lint
  ```

### 阶段 2️⃣：准备截图素材（1 小时）

按照 `screenshots/README.md` 的指南，准备以下文件：

- [ ] **2.1 拍摄 6 张应用截图**
  - `01_main.png` - 主界面
  - `02_dual.png` - 双壁纸模式

  - `04_mascot.png` - 看板娘对话
  - `05_dev.png` - 开发者名片 + 彩蛋
  - `06_editor.png` - WYSIWYG 编辑器

- [ ] **2.2 调整截图尺寸**

  - 统一调整为 **1080 x 1920 px**

- [ ] **2.3 制作 Feature Graphic**
  - 使用 Canva 或 Figma
  - 尺寸：**1024 x 500 px**
  - 内容：App Icon + "Nyanpasu - Daily Anime Wallpapers"

  - 从 `mipmap-xxxhdpi/ic_launcher.png` (192x192)

### 阶段 3️⃣：GitHub 开源（30 分钟）

按照 `RELEASE_GUIDE.md` 的 "第一阶段：GitHub 开源" 部分：

- [ ] **3.1 创建 GitHub 仓库**
  - 访问 https://github.com/new
  - 仓库名：`ACGWallpaper` 或 `NyanpasuWallpaper`
  - 设为 Public（开源）

- [ ] **3.2 初始化 Git 并推送**
  ```bash
  git init
  git add .
  git commit -m "🎉 Initial Release: Nyanpasu Wallpaper v1.1.0"
  git remote add origin https://github.com/YourUsername/ACGWallpaper.git
  git branch -M main
  git push -u origin main
  ```

- [ ] **3.3 创建 Personal Access Token**（如果需要）
  - https://github.com/settings/tokens
  - 用作推送密码

- [ ] **3.4 更新 README 链接**
  - 替换所有 `YourUsername` 为真实用户名

- [ ] **3.5 添加 Topics 标签**
  - 仓库页面 → About ⚙️ → Topics
  - 添加：`android`, `kotlin`, `wallpaper`, `anime`, `material-design`

---

### 阶段 4️⃣：生成签名 APK/AAB（20 分钟）

- [ ] **4.1 创建 Keystore（密钥库）**
  - Android Studio: `Build` → `Generate Signed Bundle / APK...`
  - 选择 "APK" → "Create new..."
  - 路径：`ACGWallpaper/nyanpasu-release.jks`
  - ⚠️ **密码必须记住！无法找回！**

- [ ] **4.2 生成 Release APK**
  - 用于 GitHub Releases
  - 输出：`app/release/app-release.apk`

- [ ] **4.3 生成 Release AAB**
  - 用于 Google Play
  - 选择 "Android App Bundle"
  - 输出：`app/release/app-release.aab`

- [ ] **4.4 备份 Keystore 文件**
  - 复制 `nyanpasu-release.jks` 到：
    - U 盘
    - 云盘（Google Drive）
    - 加密硬盘
  - ⚠️ **绝对不要提交到 GitHub！**

---

### 阶段 5️⃣：创建 GitHub Release（10 分钟）

- [ ] **5.1 创建 Release**
  - 仓库页面 → Releases → "Create a new release"
  - Tag: `v1.1.0`
  - Title: `🎉 V1.1.0 - WYSIWYG Editor Update`

- [ ] **5.2 填写 Release Notes**
  - 复制 `CHANGELOG.md` 中的 V1.1.0 内容

- [ ] **5.3 上传 APK**
  - 拖拽 `app-release.apk` 到 "Attach binaries"

- [ ] **5.4 发布**
  - 点击 "Publish release"
