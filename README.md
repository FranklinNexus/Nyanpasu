## ✨ 为什么选 Nyanpasu?

一句话：**帮你找图、裁图、设壁纸 —— 主屏和锁屏，要一样还是不一样，你说了算。**

### 🌸 双屏引擎 · Dual-Stream

| 按钮颜色 | 含义 | 效果 |
|:---:|:---|:---|
| **粉 Pink** | 同色联动 | 主屏 & 锁屏用**同一张**壁纸 |
| **蓝 Blue** | 异色双图 | 主屏 & 锁屏**各一张**，按 Pixiv 去重 |
| **灰 Off** | 关闭 | 不自动管理该屏 |

> 点预览上的 **Home / Lock** 可切换预览；所见即所得，裁切后就是最终壁纸。

### 核心功能

| | 功能 | 说明 |
|:---:|:---|:---|
| 🖼️ | **WYSIWYG 预览** | 双指缩放、拖动构图；预览框里什么样，手机上就什么样 |
| 🔍 | **Smart Tags** | **Strict 🔒** 必出你的 tag；**Soft 🎲** 20% 概率混入惊喜 |
| ☁️ | **口味滑条** | Pure / Mix / NSFW 三档，随滑条调节搜图策略 |
| 🤖 | **看板娘** | 200+ 台词，会接 tag 梗；连点 Logo **10 次**有彩蛋 🎉 |
| ⚡ | **Refresh** | 每次手动刷新都**联网换新图**；后台预取让体验更顺 |
| ⏰ | **自动换图** | 每天 **7:00**，或每 **6 / 12 / 24** 小时 |
| 🔙 | **Undo & 保存** | 最多回退 5 张历史；一键存相册 |
| 🛡️ | **隐私** | 无广告、无统计、不上传云端；图只存本机 → [Privacy Policy](PRIVACY_POLICY.md) |
| 📱 | **OEM 适配** | 针对小米 / 华为 / OPPO / vivo 等做了壁纸写入兼容 |

## 📥 下载 · Download

**30 秒上手：**

1. 打开 **[Releases → Latest](https://github.com/FranklinNexus/Nyanpasu/releases/latest)**
2. 下载 **`app-release.apk`**
3. 安装 → 授予壁纸权限 → 点 **Refresh ✨**

> **华为 / 荣耀用户：** 建议关闭「杂志锁屏」，效果更好。  
> **系统要求：** Android 7.0（API 24）及以上

## 🛠 开发者 · Developers

Kotlin · Coroutines · WorkManager · Coil · Material 3

```bash
git clone https://github.com/FranklinNexus/Nyanpasu.git
cd Nyanpasu

# Linux / macOS
./gradlew assembleDebug

# Windows（推荐，避免 R.jar 锁定）
.\scripts\build.ps1 -Test
```
